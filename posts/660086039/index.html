关键技术决策

1. CTECS OpenAPI 端点体系

发现不同服务有各自的 global 端点，不是统一的 ctecs-global.ctapi.ctyun.cn：

服务 端点 列表 API
ECS ctecs-global.ctapi.ctyun.cn /v4/ecs/volume/list (挂载查询)
EBS (云硬盘) ebs-global.ctapi.ctyun.cn GET /v4/ebs/list-ebs
CTVPC ctvpc-global.ctapi.ctyun.cn GET /v4/vpc/list
Monitor (EIP) monitor-global.ctapi.ctyun.cn GET /v4/monitor/query-eip-list
EBS Backup ebsbackup-global.ctapi.ctyun.cn GET /v4/ebs-backup/list

2. EOP-Authorization 签名要点

  • Query 参数必须按字母顺序排列才能通过签名校验
  • 签名字符串格式：headers + "\n" + sortedQuery + "\n" + sha256(body)
  • 派生密钥：ktime → kAk → kdate

3. Volume API 企业项目问题

/v4/ebs/list-ebs 的 projectID 参数是企业项目 ID，不是 cmdb_asset_account.project_id。需要在 IAM 配置企业项目转发策略，否则返回 900 错误。

4. 资源同步架构

每类资源统一模式：

  • 共同端点用 CTYunCommonInfoController 统一下发
  • 资源计数在 getResourceCountFromDB() 中添加
  • 前端卡片在 resources.vue 中配置 includesList + clickItem
