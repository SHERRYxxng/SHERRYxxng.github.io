一、功能变更摘要

1. 二类节点多账号同步

  • 改为从数据库 cmdb_asset_account 读取所有 CTYUN_LEVEL2 类型账号
  • 遍历同步，不再硬编码 AK/SK
  • 解决循环依赖：CTYunLevel2EcsServiceImpl 改用 CTYunAssetAccountMapper 代替 CTYAssetAccountService
  • InterceptorIgnoreHelper 用 try-finally 确保清理

2. VM 资源计数优化

  • 一类节点 VM 去重（3 个账号 instance_id 重叠，去重后 20 台）
  • 二类节点 VM 合并计数（20 + 179 = 199）
  • 去掉 batchId 过滤，启用 Mapper XML 去重子查询

3. 新增合并列表接口

  • POST /cmdb/ecs/cloud-server/merged-list — 合并一类+二类 VM 列表，统一分页

4. 云硬盘(EVS)同步框架

  • 建表、Entity、Mapper、Service、Task、Controller 完整搭建
  • API 端点待验证（猜测 https://evs-global.ctapi.ctyun.cn）

5. 二类节点资源发现

  • GET /cmdb/ctyun/level2/ecs/discover-resources — 检查二类账号下的 Redis/MySQL/ELB 资源
