|总字数:336|阅读时长:1分钟|浏览量:

今日工作

1. 项目运行配置

  • 项目基于 Spring Boot 3.3 + Java 17 + MyBatis-Plus + RuoYi 框架
  • 配置了本地开发环境：MySQL(localhost:3306) + Redis(Docker:6379)
  • 通过 application-local.yml 覆盖配置，不提交到 Git
  • Program arguments: --spring.profiles.active=test,local

2. 数据库配置

  • 本地库 dandelion_dev 导入完整表结构（99 张表）
  • 从测试库 172.16.1.7 同步表结构
  • 插入演示用户：手机号 13800138000 / 密码 admin123

3. 生成文档

  • project-api-documentation.md — 后端 API 接口文档（含 64+ 控制器、100+ 实体）
  • project-database-documentation.md — 数据库字段解读与映射关系（15 章）
  • CLAUDE.md — 项目编码规范、分支策略、提示词

4. 关键字段映射（从测试库查询）

  • 事件来源: 0=业务需求 1=故障修复 2=功能优化 3=安全加固
  • 紧急度/影响度: 1=低 2=中 3=高
  • 事件级别: level = 6 - (urgency + impact), P0~P4
  • 操作类型: 团队1有34种，驻场陪同随工=0106，耗材线缆入库=0206

5. Git 问题

  • 解决 GitLab 凭据频繁弹窗问题
  • 远程地址: https://gitlab.shsets.com/dev-team/dandelion/dandelion-backend.git
  • 当前分支: feature/CMDB（已推送）
  • 克隆前端项目: dandelion-frontpage（test 分支）

6. 其他

  • Docker Desktop 需保持运行（Redis 容器依赖）
  • 启动需添加 --spring.profiles.active=test,local 到 Program arguments
  • VM options 不要放 --spring.profiles.active 相关配置
