你好呀！👋

欢迎来到我的博客！这里主要记录我的生活和兴趣爱好，以及踩坑经历、个人经验总结，和我的一些主观观点。

自介

我是 Dora Honor，别名简称 SHERRY。你可以叫我 DH。

2002 年 4 月 23 日出生，江西人，家乡在江西省赣州市全南县。目前在日留学。

会说英语和日语。

在现实中话不算多，虽然我也在努力摆脱。

兴趣爱好？

喜欢动漫、游戏、漫画、绘画、唱歌、弹琴、摄影、旅游、数码、HiFi。

是双米粉（小米和米哈游）。

喜欢折腾 Windows、macOS、Linux、Android 等平台，半个耳机发烧友。

喜欢收集各种数码产品（旧款手机已有不少）。

以前喜欢解锁 BL 来折腾手机，现在没啥兴趣了。

也计划买一台 Mac 弥补没有「白苹果」的遗憾。

毕业工作存钱若干年打算提一辆 Xiaomi SU7 Max，并且自己设计痛车，自己画角色图，痛喜欢的米游角色。

和大多数人不一样，平常我并不喜欢看短视频，更喜欢看图文和长视频，以及听歌。

平常喜欢听歌，主要用索尼 NW-ZX707 播放器配天使吉米 NORA 耳机。有时也会在手机连水月雨知更鸟耳机听歌。

听下载的本地歌曲比较多，在电脑上则用 Spotify 和网易云听歌比较多。

关于本站

使用 Hexo 框架和 Butterfly 主题搭建，在 GitHub 公开源代码和在 GitHub Pages 部署，自定义了域名并用了 Cloudflare CDN 加速。

页面设置了自定义字体，且字体文件也在源代码有存放，无论在哪访问都一致。主要用的是从《原神》中提取的定制版【汉仪文黑 85W】字体，后备字体有 MiSans、Roboto、Noto CJK、各系统默认字体，确保字符都能正常显示。

卡片有半透明效果，无论日夜模式都显示正常。背景全局覆盖，一体性好。

页面点击时有烟花效果，更有个性。

评论系统用的是 Waline，支持不登录评论（但昵称和邮箱必填），自定义了背景并添加了数款表情包。

左下角有看板娘，除了插件自带的看板娘，还自己添加了芙宁娜、胡桃、爱莉希雅、流萤、昔涟 Live2D 模型看板娘。
